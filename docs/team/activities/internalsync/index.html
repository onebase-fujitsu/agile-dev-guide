Internal Sync #

概要 #

Internal Syncは、社内メンバーのみで集まって実施するミーティングです。 その日の作業の学びの共有や不明点の解消をしたり、今後の方針・進め方について相談したりします。

目的 #

社外メンバーに話しづらいことがある際に、心理的安全性を確保して共有・相談します。 また、社内報告のために学びをまとめる時間などに利用します。

実施頻度・タイミング #

  • 1回/日、30-60分程度
  • ペアワークの時間を削ることになるため、チームで合意して任意の頻度、タイミングでの実施

進め方 #

  • 参加メンバーで進め方を決める
